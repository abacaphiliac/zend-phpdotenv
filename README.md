# zend-phpdotenv
ZF2 Module that loads [vlucas/phpdotenv](https://github.com/vlucas/phpdotenv).

Module attaches a listener to EVENT_MERGE_CONFIG on ModuleManager's EventManager, before the ConfigListener merges the config.

# dependencies
* vlucas/phpdotenv
